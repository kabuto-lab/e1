Motion Icons

Lordicon Animated

1,500+ free icons Lottie JSON Trigger: hover / click / loop / morph lordicon.com ↗

Usage

Player CDN
<script src="https://cdn.lordicon.com/lordicon.js"></script>
HTML Element
<lord-icon src="https://cdn.lordicon.com/home.json" trigger="hover" colors="primary:#d4af37" style="width:48px;height:48px"> </lord-icon>
Triggers
trigger="hover" — play on hover trigger="click" — play on click trigger="loop" — infinite loop trigger="morph" — morph between states trigger="sequence" — play in order

LottieFiles Icons

Thousands of free icons dotLottie / JSON Web component lottiefiles.com ↗

Usage

Player CDN
<script src="https://unpkg.com/@lottiefiles/lottie-player@2.0.8/dist/lottie-player.js"></script>
Web Component
<lottie-player src="https://lottie.host/embed/xxx/file.json" background="transparent" speed="1" style="width:64px;height:64px;" loop autoplay> </lottie-player>
JS API
const player = document.querySelector('lottie-player'); player.play(); player.pause(); player.stop(); player.setSpeed(2);

useAnimations

200+ free animated icons Lottie JSON Open source useanimations.com ↗

Usage

Approach
Download .lottie / .json files from useanimations.com, host on your own server or CDN, then use lottie-player to render.
HTML (self-hosted)
<lottie-player src="/icons/home.lottie" background="transparent" speed="1" style="width:64px;height:64px;" loop autoplay> </lottie-player>
Categories
ACTION: home, search, settings CONTENT: user, mail, calendar NOTIFICATIONS: bell, alert, warning NAVIGATION: arrow, chevron, menu

Creattie Icons

Premium + free Lottie icons JSON / dotLottie High-quality motion creattie.com ↗

Usage

Approach
Browse creattie.com/animated-icons Download .json files, host them on your server, render with lottie-player web component.
HTML
<lottie-player src="/icons/shield-check.json" background="transparent" style="width:64px;height:64px;" loop> </lottie-player>
Style Tips
Use gold (#d4af37) color in the LottieFiles editor before export. Set loop for status indicators, no-loop for action feedback.

AnimateIcons

Free open-source SVG + CSS micro-interactions React / Vanilla animateicons.in ↗

Usage

Approach
Lightweight animated SVG icons with CSS micro-interactions. No Lottie needed — pure CSS keyframe animations.
CSS Pattern
@keyframes iconBounce { 0%, 100% { transform: translateY(0); } 50% { transform: translateY(-4px); } } .icon:hover svg { animation: iconBounce 0.4s ease; color: #d4af37; }
Pros
• No external dependencies • Tiny file size (SVG + CSS) • Full color control via CSS • Perfect for gold accent theme • Instant load, no JSON parsing
