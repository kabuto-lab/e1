🎯 О проекте

Lovnge — премиальная платформа сопровождения с акцентом на доверие, безопасность и верификацию. Не массовый каталог, а закрытая система с контролем качества и эскроу-платежами.

Ключевая идея: Только проверенные анкеты, безопасная сделка, рейтинг надёжности вместо «звёзд».

👤 Customer Journey Map (CJM)

Карта пути клиента описывает все этапы взаимодействия пользователя с платформой, его действия, мысли, боли и возможности для улучшения.

🔍 Этап 1: Осознание потребности

Действия клиента:
  • Поиск в интернете ("VIP сопровождение Москва")
  • Просмотр рекомендаций
  • Сравнение платформ
Боли и страхи:
  • Боязнь фейков и сливов
  • Недоверие к платформе
  • Риск потери денег
Возможности Lovnge: Премиум дизайн, прозрачная верификация, гарантия эскроу

🌐 Этап 2: Посещение сайта

Действия клиента:
  • Оценка визуала (чёрный + золото)
  • Просмотр главной страницы
  • Изучение каталога
  • Применение фильтров
Точки контакта:
  • Главная с ripple-эффектом
  • Каталог с 3×4 hover-сеткой
  • Карточка модели
  • Фильтры (возраст, город, tier)

📋 Этап 3: Выбор анкеты

Действия клиента:
  • Просмотр фото (10-15 шт)
  • Чтение биографии
  • Проверка верификации ✓
  • Изучение отзывов
  • Просмотр рейтинга надёжности
Критерии выбора:
  • Внешность (фото/видео)
  • Параметры (возраст, рост, вес)
  • Статус верификации
  • Рейтинг и отзывы
  • Live-статус ("в смене")

💬 Этап 4: Связь через чат

Каналы связи:
  • Telegram Bot API
  • WhatsApp Business API
  • Внутренний чат CRM
Анти-слив:
  • Скрытие номеров
  • Фильтр контактов
  • Логирование попыток
Важно: Клиент общается только с сервисом, номер исполнителя скрыт до подтверждения встречи

💰 Этап 5: Бронирование и оплата

Шаги:
  1. Выбор даты/времени
  2. Подтверждение условий
  3. Внесение оплаты
  4. Заморозка на эскроу
Способы оплаты:
  • Банковская карта
  • Криптовалюта
  • Эскроу-счёт
Гарантии:
  • Деньги замораживаются
  • Выплата после встречи
  • Возврат при отмене

🎉 Этап 6: Встреча

Сценарии:
  • Выезд к клиенту
  • Встреча в апартаментах
  • Мероприятие (дача)
Подтверждение:
  • Клиент подтверждает встречу
  • Менеджер подтверждает
  • Автоматическая выплата

⭐ Этап 7: Отзыв и лояльность

После встречи:
  • Оставление отзыва
  • Оценка рейтинга
  • Получение бонусов
Закрытый клуб:
  • Персональный менеджер
  • VIP-доступ
  • Приоритетное бронирование

🔒 Безопасность и защита

⚠️ Критические меры безопасности:
  • Анти-слив контактов (фильтр сообщений)
  • Скрытие номеров телефонов
  • Логирование попыток обмена контактами
  • Двухфакторная аутентификация (2FA)
  • Шифрование чувствительных данных

🛡️ Защита от атак

SQL Injection

  • Drizzle ORM (параметризованные запросы)
  • Валидация всех входных данных
  • Class Validator DTO

XSS Attacks

  • Экранирование вывода
  • Content Security Policy
  • Sanitization HTML

📊 Чёрные списки

ЧС Клиентов

  • Логирование всех критических действий
  • Регулярные бэкапы БД
