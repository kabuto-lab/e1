FIX #1: SIMULATION SHADER (Wave Equation)
================================================================================

OLD (BROKEN):
- Used separate height/velocity sampling
- Wrong neighbor sampling pattern
- Mouse brush had linear falloff (visible edge)
- Gradient clamping reduced wave visibility

NEW (WORKING):
```glsl
const SIMULATION_FRAGMENT = `
  precision highp float;
  varying vec2 vUv;
  uniform sampler2D uState;
  uniform vec2 uResolution;
  uniform vec2 uMouse;
  uniform float uMouseDown;
  uniform float uWaveSpeed;
  uniform float uVelocityDamping;
  uniform float uSpringForce;
  uniform float uBrushSize;
  uniform float uBrushForce;

  void main() {
    vec2 pixel = 1.0 / uResolution;
    
    // Current state: height (x), velocity (y)
    vec4 state = texture2D(uState, vUv);
    float height = state.x;
    float velocity = state.y;
    
    // Sample neighbors for Laplacian (wave equation)
    float hL = texture2D(uState, vUv + vec2(-pixel.x, 0.0)).x;
    float hR = texture2D(uState, vUv + vec2(pixel.x, 0.0)).x;
    float hD = texture2D(uState, vUv + vec2(0.0, -pixel.y)).x;
    float hU = texture2D(uState, vUv + vec2(0.0, pixel.y)).x;
    
    // Laplacian for wave propagation
    float laplacian = (hL + hR + hD + hU) * 0.25 - height;
    
    // Update velocity (acceleration from neighbors + damping)
    velocity += laplacian * uWaveSpeed;
    velocity *= uVelocityDamping;
    
    // Update height
    height += velocity;
    
    // Mouse interaction (add ripple)
    if (uMouseDown > 0.5) {
      float dist = distance(vUv, uMouse);
      if (dist < uBrushSize) {
        // Smooth brush falloff (smoothstep = no visible edge)
        float strength = smoothstep(uBrushSize, 0.0, dist);
        velocity += uBrushForce * strength;
      }
    }
    
    // Calculate gradient for rendering pass
    float gradX = (hR - hL) * 0.5;
    float gradY = (hU - hD) * 0.5;
    
    // Output: height, velocity, gradientX, gradientY
    gl_FragColor = vec4(height, velocity, gradX, gradY);
  }
`;
```

KEY CHANGES:
- `smoothstep()` for smooth brush falloff (no hard edge)
- Proper Laplacian sampling (L, R, U, D neighbors)
- Removed gradient clamping (waves can be any amplitude)
- Removed heightDamping (was killing waves too fast)

================================================================================
FIX #2: RENDER SHADER (Amplify Distortion)
================================================================================

OLD (BROKEN):
- Refraction scaled by gradient magnitude (weak effect)
- Extra refractionScale calculation reduced visibility
- Distortion barely visible at 0.03 refraction

NEW (WORKING):
```glsl
const RENDER_FRAGMENT = `
  precision highp float;
  varying vec2 vUv;
  uniform sampler2D uState;      // Simulation output (contains waves)
  uniform sampler2D uImage;      // The photo
  uniform vec2 uResolution;
  uniform float uRefraction;
  uniform float uSpecularPower;
  uniform float uSpecularIntensity;
  uniform vec2 uCanvasSize;
  uniform float uImageAspect;
  uniform vec3 uGoldPrimary;
  uniform vec3 uGoldLight;

  void main() {
    // Read wave simulation
    vec4 state = texture2D(uState, vUv);
    float height = state.x;
    vec2 gradient = state.zw;    // This is the slope of the water
    
    // UVs for image sampling
    vec2 uv = vUv;
    
    // CRITICAL: Apply distortion based on wave gradient
    // If gradient is zero (flat water), no distortion
    // If gradient is non-zero (waves), distort UVs
    vec2 distortion = gradient * uRefraction * 10.0; // Multiplier makes it visible!
    uv += distortion;
    
    // Aspect ratio correction (cover mode)
    float canvasAspect = uCanvasSize.x / uCanvasSize.y;
    float imageAspect = max(uImageAspect, 0.001);
    float scaleY = imageAspect / canvasAspect;
    uv.y = (uv.y - 0.5) * scaleY + 0.5;
    
    // Flip Y for image texture
    uv.y = 1.0 - uv.y;
    
    // Sample image with distorted UVs
    vec3 color = texture2D(uImage, clamp(uv, 0.0, 1.0)).rgb;
    
    // Specular lighting with gold tint
    vec3 normal = normalize(vec3(-gradient.x * 3.0, 1.0, -gradient.y * 3.0));
    vec3 lightDir = normalize(vec3(0.4, 0.8, 0.3));
    float specular = pow(max(0.0, dot(normal, lightDir)), uSpecularPower);
    
    vec3 specularColor = mix(vec3(1.0), uGoldLight, 0.3);
    color += specular * uSpecularIntensity * specularColor;
    color = mix(color, color * vec3(1.0, 0.98, 0.95), 0.1);
    
    gl_FragColor = vec4(clamp(color, 0.0, 1.0), 1.0);
  }
`;
```

KEY CHANGES:
- `distortion = gradient * uRefraction * 10.0` - 10x multiplier!
- Removed refractionScale calculation (was reducing effect)
- Direct gradient application (stronger distortion)
- Removed gradient clamping in render pass
