## 🔑 Ключевые файлы

### API Entry Point
**`apps/api/src/main.ts`**
- NestJS сервер
- Swagger UI на `/api/docs`
- CORS, ValidationPipe, Helmet

### Database Module
**`apps/api/src/database/database.module.ts`**
- Drizzle ORM + postgres-js
- Подключение через DATABASE_URL

### Auth Flow
**`apps/api/src/auth/auth.service.ts`**
- `register()` — создание пользователя + JWT
- `login()` — проверка пароля + JWT
- Access token: 15min, Refresh token: 7d

### State Machine (Bookings)
**`apps/api/src/bookings/bookings.service.ts`**
```
draft → pending_payment → escrow_funded → confirmed → in_progress → completed
                                 ↓              ↓           ↓
                            cancelled      cancelled   disputed → refunded
```

## 📊 API Endpoints (кратко)

### Auth
- `POST /auth/register` — регистрация
- `POST /auth/login` — вход
- `POST /auth/refresh` — обновить токены
- `GET /auth/me` — профиль (JWT required)

### Models
- `GET /models` — каталог с фильтрами
- `GET /models/stats` — статистика
- `GET /models/:slug` — профиль по slug
- `POST /models` — создать (JWT)

### Bookings
- `GET /bookings` — мои бронирования
- `POST /bookings` — создать
- `POST /bookings/:id/confirm` — подтвердить
- `POST /bookings/:id/cancel` — отменить
- `POST /bookings/:id/complete` — завершить

### Все stats endpoints
- `GET /models/stats`
- `GET /clients/stats`
- `GET /bookings/stats`
- `GET /escrow/stats`
- `GET /reviews/stats`
- `GET /blacklist/stats`
- `GET /media/stats`
