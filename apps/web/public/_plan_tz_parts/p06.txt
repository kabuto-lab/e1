Техническое задание для разработчика сайта "Премиальное сопровождение"

Шаблон пример: 
https://escortwp.com/demo/
https://escortwp.com

1. Общие требования
1.1. Цель проекта: Создание информационной платформы для размещения анкет проверенных моделей с системой безопасных сделок, внутренним рейтингом, системой подбора, и встроенной системой лояльности для клиентов и менеджеров.
1.2. Тип сайта: Информационная платформа с элементами каталога анкет, внутренним рейтингом моделей, системой безопасных сделок и личными кабинетами для клиентов и менеджеров.
1.3. Целевая аудитория: Мужчины, ищущие качественное сопровождение, модели (девушки), работающие по системе безопасной сделки.

2. Структура сайта
2.1. Главная страница
Заголовок: «Премиальное сопровождение в Москве»
Преимущества:
Надежность моделей
Безопасная сделка (эскроу-модель)
Персональные менеджеры для клиентов
Кнопка CTA (Call to Action): "Найти модель"
Блок «Как проходит встреча» с кратким описанием механики
Блок с фото моделей (обновляемые)
Ссылка на раздел "Проверенные анкеты"

2.2. Страница с анкетами моделей
Вкладки: "Все модели", "Elite"
Каждый профиль включает:
Фотогалерею (10-15 фото)
Видео-прохождение и видео-верификацию
Дату последнего подтверждения анкеты
Отзывы пользователей
Информацию о статусе (Проверена, Elite)
Подтверждения:
Паспорт проверен
Фото подтверждены
Личность подтверждена
Статистика (Количество проведенных встреч, Отмены за последние 3 месяца, Надежность)
Кнопка для перехода в личный кабинет менеджера модели
Фильтры:
Возраст
Рост
Цвет волос
Знание языка
Размер груди
Вес
Статус VIP / Elite

2.3. Страница описания безопасной сделки
Механика эскроу: как работает заморозка средств
Инструкции для клиентов и моделей
Гарантии безопасности
FAQ с часто задаваемыми вопросами

2.4. Страница с отзывами
Отзывы клиентов о моделях
Модерация отзывов
Оценка по "Надёжности" (например, 98%)

2.5. Страница для менеджеров
Личный кабинет менеджера
Функции:
Добавление анкеты модели
Просмотр заявок от клиентов
Статистика (Количество проведенных встреч, Отмены)
Начисления через безопасную сделку
Система начисления комиссий (10% с каждой встречи)
Кнопка для управления анкетами
Комиссия с каждого успеха

2.6. Страница чёрного списка анкет
Перечень фейковых анкета, жалоб, мошенничества и нарушений
Причины попадания в чёрный список
Статус анкеты: «Заблокирована», «На проверке», «Восстановлена»
Кнопка для восстановления анкеты после модерации

2.7. Закрытый клуб клиентов
Преимущества для клиентов:
Персональный менеджер
Приоритетное бронирование
Доступ к VIP моделям
Бонусы
Система лояльности
Раздел с регистрацией на клуб

2.8. Юридический раздел
Информация об эскроу-сделке
Информация о сайте как платформе (информационная платформа, не является работодателем)
Обязательность консультации с юристом для всех пользователей

3. Функциональные требования
3.1. Личный кабинет клиента
Просмотр анкет моделей
Запрос на встречу с моделью
Внесение оплаты через эскроу-систему
Просмотр истории встреч
Подписка на VIP-сервис и лояльность
Возможность отправки сообщений менеджеру модели
Возможность отправки сообщений менеджеру сайта

3.2. Личный кабинет модели
Просмотр и управление анкетой
Просмотр статистики по встречам
Возможность обновлять фото и видео
Включение статуса «Elite»
Обновление информации о статусе (доступности)

3.3. CRM система для менеджеров
Автоматическая фиксация заявок
Уведомление менеджера о новых заявках
Система сообщений с клиентами
Прозрачная статистика по каждому менеджеру
Модерация отзывов моделей
Система анкет для проверки и подтверждения данных

3.4. Система лояльности и бонусов
Персонализированные бонусы для клиентов
Возможность получать скидки за количество встреч
Подписка на VIP-пакеты

3.5. Система эскроу
Механика заморозки средств при заказе модели
Передача средств только после подтверждения встречи
Комиссия удерживается автоматически
Возможность возврата средств в случае отмены

4. Технические требования
4.1. Разработка и поддержка сайта
Платформа: WordPress или аналогичная
Адаптивность для мобильных устройств
Безопасность:
SSL-шифрование данных
Защита от SQL-инъекций
Регулярное обновление платформы и плагинов
Система хранения данных:
Фото и видео-хостинг на сервере или сторонней платформе
Эскроу-система на базе надежной платежной системы

4.2. Оптимизация
SEO-оптимизация всех страниц
Быстрая загрузка сайта
Легкость в навигации и поиске моделей

5. UI/UX дизайн
5.1. Визуальный стиль
Основные цвета: черный + золото + серебро
Минималистичный, без явной пошлости
Дорогие фотосеты моделей
Логотипы и элементы интерфейса в соответствующем стиле

5.2. Дизайн страниц
Простой и интуитивно понятный интерфейс
Легкость навигации по категориям моделей
Привлекательный дизайн для моделей и клиентов

5.3. Тестирование интерфейса
Проверка на UX/UI удобство для пользователей
Оценка пользовательской вовлеченности через тесты

6. Дополнительные требования
6.1. Модерация контента
Все анкеты и отзывы проходят модерацию
Отзывы могут быть отклонены, если они не соответствуют стандартам

6.2. Работа с клиентами
Обучение для персонала по работе с клиентами и моделями
Регулярное обновление контактных данных клиентов

6.3. Юридические вопросы
Соответствие законодательным требованиям по защите данных
Прозрачность по работе с данными и сделками

Подробное описание функционала каждой кнопки на сайте

Главная страница
Кнопка: "Найти модель"
Действие: Переход на страницу с анкетами моделей.
Путь: Главная → Страница моделей
Дополнительно: Фильтры для поиска моделей (возраст, рост, цвет волос и т.д.)

Проверенные анкеты
Кнопка: "Перейти в каталог моделей"
Действие: Открывает страницу с фильтром анкет моделей.
Путь: Главная → Страница моделей
Дополнительно: Возможность фильтрации по категориям моделей (например, по психотипам и статусу).

Как проходит встреча
Кнопка: "Подробнее"
Действие: Переход на страницу с подробным описанием процесса встречи.
Путь: Главная → Страница "Как проходит встреча"
Дополнительно: Описание шагов: от бронирования до завершения встречи, включая подробности безопасной сделки.

Конфиденциальность
Кнопка: "Узнать больше"
Действие: Переход на страницу, объясняющую принципы конфиденциальности на сайте.
Путь: Главная → Страница "Конфиденциальность"
Дополнительно: Раздел, где подробно описана защита данных клиентов и моделей.
Отзывы

Кнопка: "Оставить отзыв"
Действие: Открытие формы для добавления отзыва о встрече с моделью.
Путь: Страница моделей → Отзывы
Дополнительно: Возможность оставлять отзывы только после завершения встречи.

Вход для менеджеров
Кнопка: "Войти"
Действие: Переход на страницу авторизации менеджеров.
Путь: Главная → Вход для менеджеров
Дополнительно: Страница авторизации с полями для логина и пароля.

Система лояльности
Кнопка: "Присоединиться"
Действие: Переход на страницу с описанием преимуществ для клиентов закрытого клуба.
Путь: Главная → Страница лояльности
Дополнительно: Раздел, где клиенты могут подписаться на клуб с бонусами и приоритетным обслуживанием.
