📋 Краткое Резюме

  • testpage.html — более законченный продукт с панелью статистики, инструкциями, 12 параметрами настройки, рандомизацией и модульной архитектурой.
  • testpage_constant.html — экспериментальная версия с более сложной физической моделью волн (волновое уравнение, капиллярные эффекты), но меньшим количеством UI контролов.

⚠️ Критические Проблемы testpage_constant.html

  • Бесконечный лоадер: Исправлено добавлением onError callback + fallback timeout 3s
  • Отсутствие картинок: Использует локальные файлы из images_tst/ — требует HTTP сервера (не работает через file://)
  • Нет волн: Возможная причина — шейдер не компилируется из-за ошибки в имени uniform (texelSize вместо uTexelSize) — исправлено
  • Нет статистики: Нельзя отследить FPS и количество волн для отладки

✅ Рекомендации

  • Для локальной разработки используйте testpage.html — он стабильнее и информативнее
  • Если нужна более реалистичная физика волн — возьмите симуляцию из testpage_constant.html и интегрируйте в testpage.html
  • Для продакшена с локальными изображениями — настройте HTTP сервер (Python: python -m http.server 8080)
  • Добавьте панель статистики в testpage_constant.html для отладки производительности
