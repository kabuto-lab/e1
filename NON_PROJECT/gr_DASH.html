1. Executive Summary & Stack Alignment

Премиальная платформа сопровождения в Москве с акцентом на доверие, безопасность и контроль качества.

  • Только верифицированные анкеты
  • Эскроу-модель оплаты
  • Рейтинг надёжности вместо звёзд
  • Анти-слив контактов
  • Внутренняя CRM + строгая модерация

Обоснование стека

  • Next.js 15 App Router — SSR/SSG, Server Components, Image Optimization
  • NestJS 10 — enterprise-логика, RBAC, транзакции, Swagger
  • Drizzle ORM + PostgreSQL 16 — типобезопасность, JSONB, сложные связи
  • MinIO — presigned URL, изоляция медиафайлов

5. Technical Roadmap

Phase 1: Core & Auth (1–2 нед)

  • Monorepo (Turborepo / Nx)
  • Auth JWT + RBAC guards
  • Таблицы: users, roles, sessions
  • Dashboard layout с protected routes

Phase 2: Media & Profiles (3–4 нед)

  • MinIO + @aws-sdk/client-s3
  • Модули: media, profiles, verification
  • Presigned upload + модерационная очередь

Phase 3: Catalog & Filters (5–6 нед)

  • SSR / ISR каталог
  • Серверные фильтры + психотип-тест
  • Персональная подборка по cookie/session

Phase 4: Financials & Escrow (7–8 нед)

  • Модуль escrow + transactions
  • Webhook от платёжного шлюза
  • Динамические комиссии

Phase 5: Communication & Security (9–10 нед)

  • Анти-слив фильтр (regex + ML опционально)
  • Черные списки
  • Аудит-лог администраторов

6. Database Schema (Drizzle ORM)


// users
users: {
  id: uuid PK,
  email: text unique,
  passwordHash: text,
  role: enum('admin','manager','model','client')
}

// profiles
profiles: {
  id: uuid PK,
  userId: uuid FK,
  managerId: uuid FK (nullable),
  nickname: text,
  age, height, weight, breastSize,
  psychotype: jsonb,
  reliabilityScore: numeric(5,2),
  isVerified: boolean,
  isElite: boolean,
  liveUntil: timestamptz
}

// media_assets
media_assets: {
  id: uuid PK,
  profileId: uuid FK,
  key: text,               // minio path
  type: enum('photo','video','verification'),
  status: enum('pending','approved','rejected'),
  moderatedBy: uuid FK
}

// transactions
transactions: {
  id: uuid PK,
  clientId: uuid FK,
  profileId: uuid FK,
  amount: numeric,
  commissionRate: numeric(4,2),
  status: enum('pending','frozen','completed','disputed')
}
