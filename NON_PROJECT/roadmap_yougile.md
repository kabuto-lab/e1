# 🚀 Escort Project — Roadmap для YouGile

**Проект:** Премиальная платформа сопровождения (Escort)  
**Срок MVP:** 8-10 недель  
**Общий объём:** ~350-450 часов  
**Стек:** WordPress + JetEngine + WooCommerce + ReHub

---

## 📋 Структура для YouGile

### Колонки доски:
```
BACKLOG → TODO → IN PROGRESS → REVIEW → DONE
```

### Теги (Labels):
- `🔴 Critical` — MVP без этого невозможен
- `🟡 High` — Очень важно
- `🔵 Medium` — Желательно
- `🟢 Low` — После запуска
- `⚡ Backend` — Серверная часть
- `🎨 Frontend` — Клиентская часть
- `🔐 Security` — Безопасность
- `💳 Payments` — Платежи
- `📊 Admin` — Админка
- `👤 User` — Личные кабинеты
