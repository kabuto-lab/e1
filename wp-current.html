Рабочие процессы

Алгоритмы

1. Верификация

  1. Подача: Менеджер заполняет форму (видео, паспорт).
  2. Статус: pending (не публично).
  3. Админ: Проверка → publish + Verified или ЧС.
  4. Elite: Интервью + 0 негатива → is_elite = true.

2. Эскроу

  1. Бронь: Клиент нажимает «Забронировать» → WC заказ с model_id.
  2. Оплата: Холдирование → Processing.
  3. Встреча: Проходит.
  4. Завершение: Подтверждение → Completed → выплата 70%.
  5. Спор: Заморозка, разбор админом.

3. Анти-слив

  1. Клиент → Telegram-бот.
  2. Бот → Ticket_ID в БД.
  3. Менеджер отвечает в дашборде.
  4. Regex удаляет телефоны/emails.
  5. Бот → чистое сообщение клиенту.
  6. Лог в wp_chat_logs.
